Burmese (Myanmar) Bible

To be able to read the Myanmar Bible successfully, you may need to install some additional fonts on your computer. We therefore recommend that you download and install the fonts Padauk. This is a free Unicode fonts, which is known to work correctly with this site. For Myanmar text, you will also need to use the Graphite enabled version of the Firefox web browser.
